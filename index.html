The Web is made up of mainly HTML documents. HTML stands for hypertext markup language and works essentially as the glue for everything else on the internet. Adding CSS Style to HTML Structure

What is CSS?

CSS is the decoration of the walls, displaying the HTML in the way desired. It is applied as the law of the website with a concept called inheritance . What this means is that properties are applied to a specificed element as well as any continuing ones, essentially the entire page. CSS is a great way to avoid repetition so programmers don't have to write the same code over and over. It also avoids problems that can arise from repeated code since every code needs to be adjusted. Using CSS will just change everything in one go.

How does CSS control HTML?

With a selection, declaration, value and property. The section could be a heading tag or div class tag. This concept has been explained in this video.

There are three methods of adding CSS Styling to a Webpage:
  • Method 1: Writing CSS in the head tag of your HTML
  • Method 2: Link your HTML to separate CSS file
  • Method 3: Write your style inline with your HTML

Using Semantic Tags

Adjusting Boxes

By setting the size of the boxes to percentages, the page will adjust for changing window size.

Box Sizing

  • 1. HTML elements are just boxes with each box made up of 4 components..
  • 2. It is often hard to get the size of the box just right because of the variety of components in each box.
  • 3. There are two techniques as to how you can deal with sizing issues: setting sizes to percentages (as mentioned earlier) and setting box=sizing attibute to border-box for each element.
  • 4. Different browsers work slightly different. This causes different dispays from the same code.

Box Positioning

Using the div tag as a block element makes it difficult since it takes up the entire width of the page. Adding a rule display:flex to the appropriate CSS can override this issue.
